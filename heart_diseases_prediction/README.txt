Это был дополнительный проект, который не являлся обязательным.
Даны данные по пациентам, проходящим диспансеризацию в мед клинике. По результатам диспансеризации надо было построить ML модель, которая определит вероятность возникновения сердечно-сосудистого заболевания у пациент.

В качестве модели использовал CatBoostClassifier для которого подбирались гиперпараметры через optuna.
